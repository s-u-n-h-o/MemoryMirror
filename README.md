# 기억 저장소

---
## 프로젝트 설명
이 프로젝트는 사진을 '앨범' 형태로 분류하여 보관하고 볼수있는 웹 애플리케이션입니다.
사용자는 회원가입 후 로그인하여 앨범을 생성하고 사진을 추가,조회,수정,삭제를 할 수 있습니다.

## 주요 기능
- 사용자 인증 (로그인, 회원가입)
- 앨범 CRUD 기능 (게시글 작성, 조회, 수정, 삭제)
